# audiox

Статический сайт для ЛР: главная, форма добавления альбома, список коллекции и страница деталей.

## Страницы
- `index.html` — главная
- `form.html` — форма
- `list.html` — список (коллекция)
- `detail.html` — сведения об элементе списка (`detail.html?id=...`)

## Технологии
- Bootstrap 5 (CDN)
- Свои стили: `styles.css`
- Логика: `main.js` (данные в `localStorage`)
